# DocuSense AI

DocuSense AI is an intelligent document analysis platform that combines a React frontend, a Node.js/Express backend, and a Python/Flask machine learning service to extract insights from documents.

## Project Structure

```
DocuSense-ai/
├── frontend/      # React + Vite app
├── backend/       # Node.js + Express app
└── ml-service/    # Python + Flask app
```
